Selamat datang di repositori perjalanan belajar Python saya! 🚀

Repositori ini bukan sekadar kumpulan kode mati, melainkan sebuah jurnal dokumentasi dari proses belajar, eksperimen, dan *problem-solving* saya dalam memahami bahasa pemrograman Python dari nol. 

## 🧠 Filosofi Belajar (My Mindset)

Dalam membangun *repository* ini, saya berpegang teguh pada beberapa prinsip utama:
* **Anti Copy-Paste AI:** Semua kode akhir di dalam *repository* ini dirakit murni dari pemikiran saya sendiri. AI hanya saya pergunakan sebagai mentor/tutor untuk berdiskusi, memahami materi, dan mencari referensi, **bukan** untuk menuliskan kode instan.
* **Paham Logika, Bukan Sekadar Sintaks:** Tujuan utama saya adalah melatih *computational thinking* dan mengerti *mengapa* sebuah kode ditulis, bukan hanya *bagaimana* cara menulisnya.
* **Storytelling di Setiap Kode:** Saya percaya bahwa kode yang baik adalah kode yang memiliki cerita. Oleh karena itu, setiap tantangan logika yang berhasil saya selesaikan dilengkapi dengan *README* berisi cerita di balik layar proses *debugging*, eksperimen, dan penulisan kodenya.

## 📂 Struktur Repositori

Repository ini dibuat dengan struktur minimalis agar rapi dan mudah dinavigasi:

* 📁 **`Tantangan-Logika/`**
  Berisi *mini-project* dan tantangan sistem yang sudah berhasil saya taklukkan. Setiap sub-folder di dalamnya bersifat mandiri (berisi *file* kode utama dan cerita di balik pembuatannya).
* 📁 **`Latihan-Harian/`**
  Area "corat-coret" yang berisi potongan kode pendek, eksperimen fungsi dasar, dan catatan kecil selama proses belajar harian.
